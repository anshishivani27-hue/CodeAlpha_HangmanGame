# 🎮 Hangman Game

## 📌 Description
A simple text-based Hangman game developed using Python. The player guesses a hidden word one letter at a time with a maximum of 6 incorrect attempts.

## ✨ Features
- Random word selection
- Six incorrect guesses allowed
- Displays guessed letters
- Win/Lose messages
- User-friendly console interface

## 🛠 Technologies Used
- Python 3

## ▶️ How to Run

1. Open the terminal.
2. Navigate to the project folder.
3. Run:

```bash
python hangman.py
```
